# Changelog

## [1.4.1] - 2025-12-16

### Added

- Support for Season 11 items (Tributes, Compasses, Memories)
- Gems pickup functionality with GUI toggle
- Diobyte fork branding in GUI title

### Changed

- Crafting items now require interaction instead of walkover
- Boss items now require interaction instead of walkover
- Removed "Skip Self Dropped" feature

### Fixed

- Stuck recovery with random movement on loot timeouts
- Repeated pickup attempts on inventory full by checking item existence after interact
- Verified walkover flags based on Diablo 4 mechanics
